# Getting started

## Requirements

This repository is implemented and verified on **python 3.8.12**.

## Installation

Clone the repository and run one of the following commands.

```shell
$ make setup
```

## Outline

0. Data cleaning
1. Matplotlib
2. Seaborn
3. PCA
4. T-SNE
